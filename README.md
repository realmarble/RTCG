### Installation

Here's how to set it up

1. Clone the repo
   ```sh
   git clone https://github.com/realmarble/RTCG.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Get your data
   ```js
   npm run scrape
   ```
3. Launch it
   ```js
   npm run main
   ```
